# T-RemotEye Device Starter Kit

## Instruction
본 코드를 실행하기 위해서는 사전에 npm 및 node.js가 설치되어 있어야 하며, T-RemotEye의 테스트 서버에 접속하기 위해 방화벽 오픈이 되어야 함.<br>
**방화벽 오픈 문의 : hb.ahn@sk.com [hb.ahn@sk.com]**

1.  config.js 수정
2.  $ npm install 실행
3.  $ node device.js 실행
